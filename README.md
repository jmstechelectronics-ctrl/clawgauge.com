# ClawGauge Landing Page

Standalone one-page landing site for ClawGauge.com.

## Local preview

From this folder:

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.

## Deployment

- Static site ready for Netlify as-is.
- Publish directory: `.`
- No build step is required.

## Notes

- Uses only local assets.
- No external tracking.
- Contact form currently opens a mailto draft as the safe fallback.
- Footer credit: Built by JMS Web Design.
